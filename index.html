PROJECTS



Sign2Line

This project was created at GeauxHack 2014, LSU's inaugural hackathon, with Devan Huapaya and Eleni Mijalis. Sign2Line uses the Leap Motion to translate American Sign Language into text. Eleni and I were responsible for using the Leap Motion API to track where parts of both hands were and what signs an individual was making given that information.

Visit Site

CultureHunt

This project was created at Houston's Open Data Hackathon on the National Day of Civic Hacking with Devan Huapaya, Priscilla Lam, and Bryant Pham. This is a cultural landmark scavenger hunt. I wrote the description for each landmark and used JSON and JavaScript to create the leaderboard.
